# todo_riverpod

概要<br>
このアプリは、ユーザーがTODOアイテムを管理できるシンプルなアプリです。

・TODO一覧を表示できる<br>
・TODOはタイトル、説明、完了の有無を保持<br>
・TODOを追加できる<br>
・TODOの完了、未完了を更新できる<br>

ポイント<br>
・Freezedで不変クラスを作成<br>
・状態管理はRiverpodを使用
